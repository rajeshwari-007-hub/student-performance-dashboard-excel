# Student Performance Dashboard (Excel)

## Project Description
This project is an interactive Excel dashboard developed to analyze student performance. It uses Excel formulas and charts to present meaningful insights from student data.

## Features
- Student dataset management
- Total students summary
- Average, highest, and lowest marks
- Average attendance
- Students scoring above 80 marks
- Department-wise student distribution
- Interactive charts and dashboard

## Tools Used
- Microsoft Excel
- Formulas
- Charts
- Data Visualization

## Dashboard Insights
- Student Marks Analysis
- Department-wise Distribution
- Attendance Analysis
- Performance Summary
